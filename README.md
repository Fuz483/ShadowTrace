**ShadowTrace** — это инструмент для обнаружения скрытого шпионского ПО, которое тайно записывает аудио с микрофона и отправляет данные в интернет. Работает по принципу корреляции: процесс считается подозрительным, если он одновременно использует микрофон и имеет активные сетевые соединения.

> ⚠️ **Важно**: ShadowTrace не является антивирусом. Это инструмент для продвинутых пользователей и специалистов по безопасности для выявления аномального поведения процессов.

## 🔍 Как это работает

ShadowTrace использует трёхэтапный анализ:

### 1. Сетевая разведка (`NetworkScanner`)
- Сканирует таблицу активных TCP/UDP соединений через `psutil`
- Фильтрует локальные адреса (`127.0.0.1`, `0.0.0.0`, multicast)
- Сопоставляет каждое соединение с PID процесса

### 2. Аудио-детектор (`AudioDetector`)
- **Windows**: Анализирует загруженные DLL процессов (`memory_maps`)
- Ищет системные аудио-библиотеки: `winmm.dll`, `audioses.dll`, `wdmaud.drv` и др.
- **Linux**: Сканирует `/proc/[PID]/fd/` на наличие обращений к `/dev/snd/`

### 3. Коррелятор (`Analyzer`)
- Находит пересечение множеств PID из сети и аудио
- Фильтрует через белый список легитимных программ
- Формирует алерты с детальной информацией

### Признаки подозрительной активности
| Признак | Вес | Описание |
|:---|:---:|:---|
| Процесс в сети + аудио | 🔴 Высокий | Основной триггер |
| Исполняемый файл в Temp | 🟡 Средний | Типично для вредоносов |
| Нестандартный порт (≠443, 80) | 🟡 Средний | Обход файрвола |
| Нет обратной DNS-записи | 🟡 Средний | Скрытие инфраструктуры |
| Процесс без окна (GUI) | 🟢 Низкий | Может быть сервисом |
